# Asteroids

A classic Asteroids arcade game built with HTML5 Canvas and vanilla JavaScript.

## How to Play

Open `index.html` in a browser.

### Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Rotate and thrust |
| Space / F | Shoot |
| P | Pause |
| Enter | Start / Restart |

### Scoring

| Asteroid | Points |
|----------|--------|
| Large | 20 |
| Medium | 50 |
| Small | 100 |

An extra life is awarded every 10,000 points. High score is saved in localStorage.
